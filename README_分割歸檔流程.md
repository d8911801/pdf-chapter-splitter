# PDF 電子書章節分割與歸檔工作流程

## 適用場景

從掃描版 PDF 電子書中按章節分割出析出文獻，命名後歸檔至研究目錄。

## 前置條件

- Python 3.x
- `pip install PyPDF2`

## 三步流程

### 第一步：提取書籤（目錄大綱）

```python
import PyPDF2
reader = PyPDF2.PdfReader("書名.pdf")
for item in reader.outline:
    page = reader.get_destination_page_number(item) + 1
    print(f"第{page}頁 | {item.title}")
```

若書籤為逐頁序號（如「1」「2」「3」…），表示該書無章節書籤，需先 OCR 目錄頁圖片，手動圈定章節邊界。

### 第二步：編輯配置並執行分割

編輯 `split_ebook_chapters.py` 中的 `BOOKS` 列表，填入：
- `book_title`：書名
- `author`：作者
- `path`：原始 PDF 絕對路徑
- `chapters`：`[(起始頁碼, 章節標題), ...]`  頁碼為 PDF 顯示頁碼（1-based）

執行：
```bash
python split_ebook_chapters.py
```

輸出目錄預設為 `桌面\上博楚簡_分割章節`。

### 第三步：歸檔

1. **歸入「依篇目分類」**：
   - 將篇名涉及上博楚簡具體篇章（如《緇衣》《民之父母》《孔子詩論》等）的 PDF，
     複製到 `參考文獻\依篇目分類\` 下對應子資料夾。
   - **若無對應子資料夾**，則在該目錄下建立新資料夾，命名格式參照現有編號（如「24恒先」「31曹沫之陣」）。
   - 複製前檢查目標資料夾是否已有同名檔案，避免覆蓋。

2. **歸入「書籍」目錄**：
   - 為每本書建立 `作者：書名` 子資料夾於 `電子書\思想學術\書籍\` 下。
   - 將全書 PDF + 所有析出章節 PDF 複製進去。
   - 若資料夾已存在，直接複製檔案（不刪除原有檔案）。

## 命名規則

| 情況 | 格式 | 示例 |
|------|------|------|
| 章節作者 = 書名作者 | `作者：《篇名》，《書名》.pdf` | `丁四新：《第一章 竹书《孔子诗论》的诗学思想》，《上博楚竹书哲学文献研究》.pdf` |
| 章節作者 ≠ 書名作者 | 暫停，詢問使用者 | — |

## 注意事項

- 全書 PDF 和桌面分割檔案均為**複製**（非移動），原始檔案保留不動
- 歸檔前檢查目標目錄是否已有同名檔案，避免覆蓋
- Windows 檔名非法字元（`: * ? " < > | / \`）自動替換為全形版本
